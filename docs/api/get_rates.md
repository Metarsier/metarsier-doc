# 获取汇率
URL: `/api/common/get-rates`  
Method: `POST`

> 参数   

| 参数名        | 类型         | 是否必须   | 描述                                |
| ------------ | ----------- | --------- | ---------------------------------- |
| refUserId  | string      | YES       | 用户ID（商户内部）                    |

> Request   

```json
{
    "refUserId": "13800138000"
}
```
